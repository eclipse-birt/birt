[Non-API] This package contains classes that implement the JDBC-ODA Bridge Driver.

ODA (Open Data Access) is the standard data extension protocol for providing JDBC data access to BIRT. The JDBC-ODA Bridge Driver translates ODA API requests to JDBC calls, thus enabling access to JDBC data sources in BIRT reporting.

Note: The implementation classes are not public APIs.
Backward compatibility support in future releases is not guaranteed.
