Designing a Multipage Report

Most reports display and print on multiple pages. When designing a report, consider how a multipage report displays and prints. For example, you can design a report so that page breaks occur at logical places. For a report that contains a series of subreports, you can specify page breaks to start each subreport on a new page.

You can also design a page layout that enhances the report’s appearance and usability. A page layout specifies the style of the pages that display report data. For example, you can specify that all pages display the report title in the top left corner, the company logo in the top right corner, a page number in the bottom right corner, and a report-generation date in the bottom left corner. You can also design special effects, such as a watermark that appears in the background. To design a page layout, customize the report’s master page.
