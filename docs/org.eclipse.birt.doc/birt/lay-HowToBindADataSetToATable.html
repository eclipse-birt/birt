How to bind a data set to a table

1  
In the layout editor, hover the mouse pointer over the bottom left corner of the table until the Table tab appears, then choose the tab.
2  
Choose the Binding tab at the top of Property Editor. The Data Set field shows either the name of the data set that is currently bound to the table or None if no data set is bound to the table.
3  
From the Data Set drop-down list, select a data set. BIRT Report Designer binds the data set to the table. It also creates a column binding for each data set field. Elements placed in the table can now access all the fields in the data set.
