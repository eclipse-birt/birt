How to drop a theme from a report design

To drop a theme, change the Theme property of the report root item.

1  
In the Outline view, select the report root item, as shown in Figure 19-2. The report designâ€™s properties appear in Property Editor.

Figure 19-2  Report root item selected
2  
In Property Editor, select None from the list in Themes, as shown in Figure 19-3.

Figure 19-3  Dropping a theme in Property Editor
Now, the theme that the report design uses is None, which means that it uses only its own styles or styles from an included CSS file, not styles from a theme in a library.
